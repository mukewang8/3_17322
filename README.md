<h3>课程介绍：</h3>
<p>通过该课程的学习，学员能学习针对一个网站的完整测试过程及思路，极大的帮助其在测试过程中成长</p>
<p>本课程小编利用自建网站，利用工具和手工两个方面将一个网站进行了测得的测试，并制作成课程。在测试的过程中涉及到的知识点进行了一一的介绍，课程中使用到的工具也会以附件的形式放在课程中，方便大家学习。</p>
